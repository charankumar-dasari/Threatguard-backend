# 🛡️ ThreatGuard Backend

A Spring Boot based REST API for URL Threat Detection and Security Monitoring.

ThreatGuard Backend provides secure authentication, role-based authorization, URL threat analysis, scan history management, dashboard analytics, and administrative controls through a RESTful architecture.

---

## 🚀 Features

### Authentication & Security
- User Registration
- User Login
- JWT Authentication
- BCrypt Password Encryption
- Role-Based Access Control (Admin/User)
- Protected REST APIs

### URL Threat Detection
- URL Analysis
- Threat Classification
- Threat Score Generation
- Malicious URL Verification

### User Features
- Scan URLs
- View Scan History
- Dashboard Statistics
- Activity Tracking

### Admin Features
- User Monitoring
- Malicious URL Management
- Threat Monitoring Dashboard
